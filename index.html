Backup

Two minutes, once — then every workout, meal, and review auto-saves to your own private GitHub repo.

  1. Create the repo → name: coach-backup · visibility: Private · hit Create repository. Ignore everything else on that page.
  2. Create the token → any name · Expiration: 1 year · Repository access: Only select repositories → pick coach-backup · open Repository permissions → set Contents to Read and write · Generate token · copy it.
  3. Paste it below with the repo as yourusername/coach-backup, then hit test.

The snapshot never contains your API key or this token. If it fails, the message below the button tells you exactly which step to redo.
